IPFUnpacker

IPFUnpacker decrypts IPF files in the KROBT client.

IPFUnpacker

Usage : unpack.exe <IPF file>

Be careful when using it : it will erase the old IPF and use the same file for decrypting it, so make sure to backup your IPF files somewhere before decrypting them.

Once decrypted, the IPF is readable by traditional tools (such as IPF Suite).
